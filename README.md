Front End:

Technologies used :

	2.) React Bootstrap 5
	3.) Redux Saga

Commands:

	npx create-react-app@5.0.0 employee-management-ui
	npm i redux-devtools-extension
	npm i react-router-dom
	npm i bootstrap@5.1.3
	npm i node-sass
	npm i redux-saga	
	npm i axios
	
	npm audit fix --force
	
	cd employee-management-ui
	npm start
